Settings

Google Drive sync

Google Drive sync (primary). For Google Drive backup, click Sign in with Google below. Your library syncs automatically to game-vault-backup.json. API keys always stay on this device, and health records stay local unless you explicitly enable health sync under System, recovery & diagnostics. One-time setup: open console.cloud.google.com → create a project → enable the Google Drive API → configure the OAuth consent screen (External, then Publish) → Create Credentials → OAuth client ID → Web application → add https://sinuksml.github.io under Authorized JavaScript origins → copy the Client ID below.

GameVault Drive storage: not connected.

Android TV phone login. If normal Google sign-in does not open on the TV browser, create a Google OAuth client with application type TVs and Limited Input devices, paste it below, then use the QR login. Scan the QR on your phone, approve access, and the TV will connect to the same Drive backup.

Games, movies & TV data services

Live internet updates (optional). Get a free API key from rawg.io/apidocs (sign up → Get API Key), paste it below, and the app can fetch cover art, scores, autocomplete and live lists for Upcoming Releases and Discover from the RAWG games database. Without a key, the built-in curated lists still work fully offline.

Movies & TV Shows sections (TMDB + IMDb). These sections pull live data from TMDB — sign up free → Settings → API → copy the API Key (v3 auth). For exact IMDb ratings where available, also get a free key from OMDb (activate the link they email you). Paste both below.

BiglyBT & Plex

BiglyBT secure dashboard. Paste your HTTPS Cloudflare Worker URL. The streamlined Native Dashboard is recommended; the original Web UI remains available from the BiglyBT mode switch. The raw http:// public-IP address cannot be embedded because GameVault uses HTTPS.

Plex library. Enter an owner X-Plex-Token and either your secure Plex Media Server URL or use Discover Server. The token stays only in this browser and is never included in Drive/JSONBin backups. For deletion, enable Settings → Server → Library → Allow media deletion in Plex. Use an HTTPS plex.direct connection when opening GameVault from GitHub Pages.

JSONBin fallback

JSONBin sync (optional fallback). Used only when Google Drive is not connected. Get a free account at jsonbin.io → open API Keys → copy your Master Key and paste it below on this device, then Save. Every change you make here now saves to the cloud automatically a couple of seconds after you type — no button to press. On your other device (phone or PC), open the app, paste the same Master Key, then copy the Bin ID shown here into that device's Settings too, and Save. From then on both devices silently stay in sync: whichever has the newest change wins, checked every time the app opens. The Push/Pull buttons in the ⋯ menu still work if you ever want to force a sync manually.

System, recovery & diagnostics

Version policy: patches such as 1.1.1 are fixes, minor versions such as 1.2.0 add features, and major versions such as 2.0.0 represent substantial redesigns or incompatible behavior.

TV display size

Health sync is off by default. If enabled, health records are included in your normal Drive or JSONBin backup. Finance records are always stored as a separately PIN-encrypted envelope; the PIN is never saved or uploaded. Use Encrypted backup for a password-protected portable copy of the remaining vault.

GameVault keeps up to eight local recovery points and up to five daily Drive history copies. Diagnostics never include passwords, OAuth tokens, Plex tokens, or API keys.
