GAMEVAULT

Settings

Google Drive sync

Automatic backup and cross-device synchronization. Your library is stored in game-vault-backup.json in Google Drive. Health records remain local unless cloud sync is enabled below.

GameVault Drive storage: not connected.
Google Drive setup help

Open Google Cloud credentials, enable Google Drive API, configure the OAuth consent screen, create a Web application OAuth client, and add https://sinuksml.github.io as an authorized JavaScript origin. The native Android TV app manages its own TV login.

Games, movies & TV data services

RAWG supplies current game releases, artwork and recommendations.

TMDB and OMDb supply movie and TV artwork, release information and IMDb ratings.

Data-service setup help

Create a RAWG key at rawg.io/apidocs, a TMDB v3 key under TMDB settings, and an OMDb key at omdbapi.com.

BiglyBT & Plex

BiglyBT native dashboard. Connect through the HTTPS Cloudflare gateway.

Plex library. Connect using a secure Plex server URL and owner token.

Plex and BiglyBT setup help

Use an HTTPS plex.direct connection for Plex. Find the owner token in the Plex token guide. For BiglyBT, the raw public-IP address cannot be embedded because GameVault uses HTTPS.

System, recovery & diagnostics

Version policy: patches such as 1.1.1 are fixes, minor versions such as 1.2.0 add features, and major versions such as 2.0.0 represent substantial redesigns or incompatible behavior.

Encrypted credential sync

Protect API keys and Plex/BiglyBT access with a separate passphrase before storing them in Drive. Enter the same passphrase once on each trusted device.

Health sync is off by default. If enabled, health records are included in your Google Drive backup. Finance remains protected by its dedicated PIN and Face ID lock. Use Encrypted backup for a password-protected portable copy of the remaining vault.

GameVault keeps up to eight local recovery points and up to five daily Drive history copies. Diagnostics never include passwords, OAuth tokens, Plex tokens, or API keys.
